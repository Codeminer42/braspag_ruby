# Braspag

Provides an easier Ruby integration with the Braspag payment gateway.

## Installation

Add this line to your application Gemfile:

```bash
gem 'braspag'
```

And then execute:

```bash
$ bundle
```

Or install it yourself as:

```bash
$ gem install braspag
```
